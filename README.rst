vim
===

Installation
============

Using git you can have the full master branch: ::

    git clone https://github.com/ScorpiusZ/vim.git 
    mv -u vim  ~/.vim
    cd ~/.vim
    git submodule init
    git submodule update

This may take some time.


Usage
=====

Using my vim config: ::

    touch ~/.vimrc
    add a line below
    source ~/.vim/vimrc

then you will have exactly same vim envirment like me .
Hope you like it.
